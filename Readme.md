# 🚀 Lista De Tarefas

Esta é uma aplicação de Lista de Tarefas/Compras desenvolvida para praticar lógica de programação e manipulação de eventos no navegador.

## 🛠 Tecnologias
- HTML5
- CSS3 (Flexbox e Design Responsivo)
- JavaScript

## ⚙️ Funcionalidades
- [ Usuario digita a tarefa] Funcionalidade 1 
- [ Clica no botão para gravar ] Funcionalidade 2 
- [ O Usuario pode apagar a tarefa quando quiser ] Funcionalidade 3 

## 🧠 Desafio Técnicos 

O maior diferencial deste projeto foi a manipulação dinâmica de elementos. Em vez de apenas esconder elementos pré-existentes, o código:

1. Cria novas tags (como <li> e <button>) via JavaScript.

2. Atribui classes e atributos em tempo de execução.

3. Utiliza o método .appendChild() para renderizar os novos nós na árvore do DOM de forma estruturada.
